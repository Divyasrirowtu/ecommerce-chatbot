# Setup Instructions

1. Clone the repo:
git clone https://github.com/Divyasrirowtu/ecommerce-chatbot.git
cd ecommerce-chatbot

2. Create virtual environment:
python -m venv venv
venv\Scripts\activate

3. Install dependencies:
pip install -r requirements.txt

4. Run chatbot:
python chatbot.py

5. View results:
eval/results.mds
